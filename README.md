# GitHub Profile Analyzer

A high-performance, visually polished Swiss-Modern web application designed to analyze and visualize GitHub user profiles in deep, multi-dimensional detail. Featuring premium data visualizations, real-time repository health checks, and a comprehensive activity heatmap, this full-stack tool turns GitHub activity metadata into structured, actionable developer intelligence.

---

## 🚀 Key Features

### 1. Developer Tier & Profile Intelligence
- **Intelligent Tier Classification**: Automatically ranks users into categories (e.g., *Elite Contributor*, *Active Developer*, or *Casual Coder*) based on their calculated average repository health and activity metrics.
- **Unified Metadata Dashboard**: Displays high-level stats like total repositories, total stars, total forks, and follower counts in a clean, elegant typographic grid.

### 2. Multi-Dimensional Recharts Visualization Suite
- **Skill Radar Chart**: Maps developer expertise across 7 predefined technology disciplines (*Frontend, Backend, Systems, Mobile, AI/ML, DevOps, and Data*).
- **Tech Stack Breakdown**: Donut chart outlining proportional categories of technology stacks with a custom dark-mode tooltips system.
- **Top Languages**: High-density bar charts displaying the top programming languages.
- **Commit Trend Graph**: Area chart tracing recent daily commit velocity over time.
- **Productivity Timeline**: Line chart detailing commit density by hour of the day (24-hour cycle) to showcase working habits.
- **Repository Creation Timeline**: Historical bar chart mapping open-source repository creations across calendar years.

### 3. Professional Contribution Matrix
- **90-Day Heatmap (12 Weeks)**: A fully modeled representation of user contribution frequency.
- Includes Monday/Wednesday/Friday day alignment and dynamic short-month headers.
- Multi-tier color opacity blocks (ranging from subtle gray backdrops up to vibrant brand orange `#FF5C00`) to depict code velocity.
- Interactive hover tooltips displaying date-specific contribution counts.
- Displays metrics for current streaks, longest streaks, and peak productivity hours.

### 4. Repository Health Assessment Cards
- **Interactive Repository Cards**: Outlines top-performing code repositories with translation offsets (`hover:-translate-y-2`) and solid shadow borders.
- **Health Score Algorithm**: Scores each repository out of 100 on documentation readiness, active maintenance, issue ratio benchmarks, and general modern conventions (licenses, wikis, page usage, and active updates).

---

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript, Vite 6, Tailwind CSS v4, Lucide React (for premium typography and icons).
- **Charts & Graphics**: Recharts (fully customized theme to match Swiss-Modern typography guidelines).
- **Backend & Server-side**: Express (serving as the application environment and developer proxy).
- **Developer Engines**: TSX, ESBuild (CJS bundling of server entry point), and custom TypeScript transpilers.
